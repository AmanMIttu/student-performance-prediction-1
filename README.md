# Student Performance Prediction

## Overview
This project predicts whether a student will pass or fail based on demographic and academic attributes using machine learning.

## Techniques Used
- Data Cleaning and Preprocessing
- Label Encoding
- Feature Engineering
- Random Forest Classification

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn

## Outcome
The model analyzes academic patterns and identifies factors influencing student performance.
